Description

Combinators using CVars. These provide many of the helpful functions found in Control.Concurrent.MVar, but for CVars. Note that these do not in general mask exceptions, and are not atomic, being implemented in terms of the primitives in the ConcFuture and ConcCVar typeclasses.

Binary semaphores

A common use of CVars is in making binary semaphores to control mutual exclusion over a resource, so a couple of helper functions are provided.

lock :: ConcCVar cvar m => cvar () -> m ()Source

Put a () into a CVar, claiming the lock. This is atomic.

unlock :: ConcCVar cvar m => cvar () -> m ()Source

Empty a CVar, releasing the lock. This is atomic.
